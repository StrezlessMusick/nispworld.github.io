Title
Casualty Rate Estimate Profile
Description

A set of CRE settings representing a particular scenario is called a CRE Profile. Each profile contains data sets relating to the Operational Scenario and the Force Structure. The force structure includes force strength, level of command, parent unit, subordinate units, PAR. The Operational Scenario is set of factors/elements that affect an operation (e.g. operation type, operational form, environment conditions, environmental health factors, epidemiological assessments, threat assessment)
