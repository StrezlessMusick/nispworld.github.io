ID: fmn1-basic-text-based-collaboration-profile

Title
Basic Text-based Collaboration Service Profile 1.0
Description
The Basic Text-based Collaboration Profile provides standards and guidance to establish a basic near-real time text-based group collaboration capability (chat) for time critical reporting and decision making in military operations.

Obligation: Optional, Lifecycle: Current

Bidirectional Server-to-Server Connections may be supported, i.e. stanzas are sent and received on the same TCP connection.

  • XEP-0228: Bidirectional Server-to-Server Connections (XMPP XEP-0288:2010)

Obligation: Mandatory, Lifecycle: Current

The following standards are required to achieve compliance for an XMPP Server and an XMPP Client dependent upon the categorisation of presenting a core or advanced instant messaging service interface.

  • XEP-0004: Data Forms (XMPP XEP-0004:2007)
  • XEP-0030: Service Discovery (XMPP XEP-0030:2008)
  • XEP-0045: Multi-User Chat (XMPP XEP-0045:2012)
  • XEP-0049: Private XML Storage (XMPP XEP-0049:2004)
  • XEP-0050: Ad-Hoc Commands (XMPP XEP-0050:2005)
  • XEP-0054: vcard-temp (XMPP XEP-0054:2008)
  • XEP-0092: Software Version (XMPP XEP-0092:2007)
  • XEP-0096: SI File Transfer (XMPP XEP-0096:2004)
  • XEP-0114: Jabber Component Protocol (XMPP XEP-0114:2012)
  • XEP-0115: Entity Capabilities (XMPP XEP-0115:2008)
  • XEP-0203: Delayed Delivery (XMPP XEP-0203:2009)
  • XEP-0220: Server Dialback (XMPP XEP-0220:2013)

Obligation: Mandatory, Lifecycle: Current

The following standards are the base IETF protocols for interoperability of chat services.

  • Extensible Messaging and Presence Protocol (XMPP): Core (IETF RFC 3920:2004)
  • Extensible Messaging and Presence Protocol (XMPP): Instant Messaging and Presence (IETF RFC 3921:2004)
