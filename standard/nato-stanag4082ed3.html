Title
Adoption of a Standard Artillery Computer Meteorological Message

Applicability

The aim of this STANAG is to define a computer meteorological message and to standardize the number of its information digits and their meanings for use by any artillery service of any country. This will make it possible for all armed forces to understand and be able to use an unweighted artillery meteorological message which is compatible with computers.
