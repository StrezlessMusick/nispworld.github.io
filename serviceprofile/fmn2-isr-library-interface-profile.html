Title
ISR Library Interface Profile
Description
The ISR Library Interface is the standard interface for querying and accessing heterogeneous product libraries maintained by various nations.

Standards

Obligation: Mandatory, Lifecycle: Current

The following NATO standards are mandated for interoperability of ISR libraries.

  • NATO Secondary Imagery Format (NSIF) - AEDP-04 Edition 2 (NATO STANAG 4545 Ed 2:2013)
  • NATO Ground Moving Target Indicator(GMTI) Format - AEDP-07 Edition 2 (NATO STANAG 4607 Ed 3:2010)
  • NATO Standard ISR Library Interface (NSILI) (NATO STANAG 4559 Ed 3:2010)
  • NATO Digital Motion Imagery Standard - MISP-2015.1 (NATO STANAG 4609 Ed 4:2016)

Obligation: Mandatory, Lifecycle: Current

Note implementation of STANAG 5525 in the context of the ISR Library Interface Profile is limited to the definition of unique keys that could be used to unambiguously refer to an external information object that is modelled in accordance with STANAG 5525.

  • Joint C3 Information Exchange Data Model (JC3IEDM) (NATO STANAG 5525 Ed 1:2007)

Obligation: Mandatory, Lifecycle: Current

The following international standards are mandated for interoperability of ISR libraries.

  • Representation of Names of Languages Part 2: Alpha-3 (ISO 639-2:1998)
  • Information technology -- Metadata registries (MDR) -- Part 3: Registry metamodel and basic attributes (ISO/IEC 11179-3:2013)
  • Image Processing and Interchange (IPI) - Functional Specification - Part 5: Basic Image Interchange Format (BIIF) (ISO/IEC 12087-5:1998)
  • Information technology -- Open Distributed Processing -- Interface Definition Language (ISO/IEC 14750:1999)
