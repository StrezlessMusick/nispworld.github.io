Title
Collection Task List (CTL)
Description

A Collection Task List (CTL) is list of prioritised and unfulfilled ISR collection and exploitation requirements for non-organic ISR capabilities at an echelon. This is also sometimes referred to as a Collection Requirements List (CRL). Once ratified by the Joint Collection Management Board, the CTLs will also include details of suitable non-organic ISR collection and exploitation capabilities to satisfy each requirement. The Collection Task List is part of the Collection and Exploitation Plan (CXP).
