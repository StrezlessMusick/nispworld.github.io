NATO Standard Bar Code Symbologies

Applicability

A bar code is an array of parallel lines of varying width used to represent data. The bar code is designed to be optically read by a data-capturing device. Traditional one-dimensional bar codes use the barâ€™s width as the code, and typically encode just an identification or account number. Two-dimensional systems hold up to 1800 characters in an area the size of a postage stamp.
