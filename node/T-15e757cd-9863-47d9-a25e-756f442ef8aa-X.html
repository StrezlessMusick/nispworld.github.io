Title
Theatre Logistics Architecture
Description

The Theatre Logistics Architecture contains the principal aspects of logistics in theatre. This architecture includes Strategic Lines of Communication (LOC); Access to the theatre and entry points, including APODs and SPODs; Intermediate Staging Bases (ISB), if required; Possibilities for Host Nation Support (HNS); and Theatre LOC.
