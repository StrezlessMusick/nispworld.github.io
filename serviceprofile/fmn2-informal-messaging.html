Informal Messaging Profile 2.0
Description
The Informal Messaging Profile provides standards and guidance for SMTP settings and the marking and classification of informal messages.

Standards

Obligation: Mandatory, Lifecycle: Current

Regarding Simple Mail Transfer Protocol (SMTP), the following standards are mandated for interoperability of e-mail services within the Mission Network.

  • Simple Mail Transfer Protocol (IETF RFC 5321:2008)
  • SMTP Service Extension for Message Size Declaration (IETF RFC 1870:1995)
  • SMTP Service Extension for Remote Message Queue Starting (IETF RFC 1985:1996)
  • SMTP Service Extension for Returning Enhanced Error Codes (IETF RFC 2034:1996)
  • SMTP Service Extension for Command Pipelining (IETF RFC 2920:2000)
  • SMTP Service Extension for Secure SMTP over TLS (IETF RFC 3207:2002)
  • SMTP Service Extension for Delivery Status Notifications (IETF RFC 3461:2003)
  • Message Disposition Notification (IETF RFC 3798:2004)
  • SMTP Service Extension for Message Tracking (IETF RFC 3885:2004)
  • SMTP Service Extension for Authentication (IETF RFC 4954:2007)
