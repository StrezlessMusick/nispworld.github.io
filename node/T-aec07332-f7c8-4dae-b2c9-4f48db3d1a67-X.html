Title
MEL/MIL
Description

The Main Events List and Main Incidents List (MEL/MIL), the main tool (normally a database) for the EXCON to control the exercise, is maintained by EXCON and it is structured on the main events developed to support achievement of the exercise objectives. Each main event will have one or more incidents that are presented to the training audiences by means of injections. The MEL/MIL should encompass the complete timeline of the exercise and, at ENDEX, be updated to include all dynamic and unscripted events, incidents and injections utilised during the exercise conduct.
