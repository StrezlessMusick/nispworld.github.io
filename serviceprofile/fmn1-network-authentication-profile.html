Title
Network Authentication Profile
Description
The Network Authentication Profile provides standards and guidance for to provide strong authentication for client/server applications by using secret-key cryptography on the basis of the Kerberos authentication protocol

Strong authentication using Simple Authentication and Security Layer (SASL).

  • The Kerberos Version 5 Generic Security Service Application Program Interface (GSS-API) Mechanism: Version 2 (IETF RFC 4121:2005)
  • Simple Authentication and Security Layer (SASL) (IETF RFC 4422:2006)
  • Anonymous Simple Authentication and Security Layer (SASL) Mechanism (IETF RFC 4505:2006)
  • The PLAIN Simple Authentication and Security Layer (SASL) Mechanism (IETF RFC 4616:2006)
  • The Kerberos v5 Simple Authentication and Security Layer (SASL) Mechanism (IETF RFC 4752:2006)
