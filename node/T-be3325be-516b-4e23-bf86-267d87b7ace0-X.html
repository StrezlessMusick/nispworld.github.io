Title
Vessel of Interest (VOI) List
Description

A list of vessels of interest shall be defined by the following information * name of the list * description of the interest of the list * date of validity of the list * vessels. A vessel is defined by the following information * Identification (name, IMO) * Characteristics * Speed Capabilities * Position (past, current, future) * Predicted route * Elint * Crew * Damage * Images * Assignments * Events * Files * Links.
