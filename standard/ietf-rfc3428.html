Title
Session Initiation Protocol (SIP) Extension for Instant Messaging

Applicability

This document proposes the MESSAGE method, an extension to the Session Initiation Protocol (SIP) that allows the transfer of Instant Messages. Since the MESSAGE request is an extension to SIP, it inherits all the request routing and security features of that protocol.
