Event Overlay
Description

A Event Overlay is a guide for collection planning. The event template is developed by comparing the analyses depicted on the situation templates for each of the COAs that the adversary is capable of executing. The event template depicts the named areas of interest where activity, or its lack of activity, will indicate which course of action the adversary has adopted.
